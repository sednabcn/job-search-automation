How It Works

🔍 Automatic Discovery
Jobs are discovered from LinkedIn, Glassdoor, Reed, and Indeed via job-discovery-worker.yml
✨ CV Optimization
Resumes are tailored for each application using Resume-Cover-Letter_Optimizer.py
🚀 Auto-Apply
Top matches can be automatically applied to when enabled
📊 Tracking
All applications are logged in applications.json
⏰ Deadlines
Application deadlines are monitored in application_deadlines.json
🎯 Orchestration
The job-manager-center-platform.yml workflow coordinates all automation
